# Helpdesk Ticketing & IT Support Operations Lab

## Overview
An osTicket-based helpdesk lab built on a Ubuntu Server LAMP stack (Apache, MariaDB, PHP), integrated with an Active Directory environment to simulate real enterprise IT support operations.

## What Was Configured
- Ubuntu Server with full LAMP stack built from scratch
- osTicket installed and configured end-to-end
- SLA plans, departments, help topics, and agent roles
- Active Directory integration — tickets tied to real domain user accounts
- Real helpdesk tickets worked from open to close — password resets, software requests, hardware issues, employee offboarding
- Ticket escalation between Tier 1 and Systems teams
- Knowledge base articles created for recurring issues

## Tools Used
- VMware Workstation Pro
- Ubuntu Server
- Apache
- MariaDB
- PHP
- osTicket
- Active Directory
- Windows 11
